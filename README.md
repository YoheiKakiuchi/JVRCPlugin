Referee support plugin and customizers for JVRC ( Japan Virtual Robotics Challenge )

$ cd choreonoid/ext
$ git clone https://github.com/jvrc/JVRCPlugin
$ cd ..
$ ccmake .
<turn on BUILD_JVRC_PLUGIN option>
$ make
$ make install
